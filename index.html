Content-Aware Image Resizer

Built an image resizing program that uses a seam carving algorithm for content-aware resizing. The algorithm finds and removes seams in the image that pass through the least important pixels.

Utilized pointers, strings, streams, IO, and abstract data types; handled errors and extensively tested and debugged code.

ParseCal

MacOS Menu Bar application to automate the process of creating Google Calendar events. This project is designed to save users time and energy, especially in situations where they have a handful of events to create. It also reduces human error when creating such events.

Built on Python using Rumps for the GUI, and Google Gemini and the Google Calendar API for the backend.

  • Python
  • Rumps
  • Google Calendar API
